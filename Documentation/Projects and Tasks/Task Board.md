# Task Board

The task board is the main place to inspect and update work. Tasks are grouped by status and can be filtered by user, status, phase, priority, project, and search text.

## Daily workflow

1. Start on Tasks.
2. Use the User filter to focus on assigned work or switch to all users.
3. Open a task row to inspect details in the review pane.
4. Update status, phase, priority, or assignee when your permission level allows it.
5. Toggle subtasks as work is completed.
6. Add comments or subtasks from the composer.

## Review panes

Task rows open review panes for task details. The pane shows description, metadata, subtasks, review media versions, comments, followers, and composer actions. Up to three tasks can be inspected side by side through multi-select behavior.

## Subtasks

Subtasks track the practical pieces of a task. When every subtask is complete, the parent task moves to Done. The app also provides an Undo to WIP action so the behavior can be tested without losing the task.

## Archiving

Tasks and subtasks are archived instead of permanently deleted. Archive actions are available through visible buttons, context menus, and keyboard Delete or Backspace behavior for selected rows.
